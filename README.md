## 🎯 KPIs Tracked

| KPI | Description | Frequency |
|-----|-------------|-----------|
| Total Failures | Count of failures per period | Daily/Weekly/Monthly |
| Failure Rate by Team | % of failures per team vs total | Weekly/Monthly |
| Top Reps | Representatives with most occurrences | Weekly/Monthly |
| Week-over-Week Variation | % change in failures vs previous week | Weekly |
| Failure Type Distribution | Breakdown by system error, connectivity, hardware, etc. | All |
